This Project

This project began with a simple question, “Can you create a program that I can plug data into that will predict what is wrong with the machinery I’m working on?” That question was asked by Jim’s father. After creating our best attempt at solving that problem in the six week timeline we had, we began to see the potential for it.

Because of that, we decided to take this on as our final project for our University’s graduation requirements. Both because it was satisfactory for the project, and because we wanted to see how far toward our combined vision we could take it.

While we have a vision in mind for this project and the direction that it is taking, which will be outlined below, it is by no mean set in granite. If you have any suggestions or comments regarding this project and its direction, please utilize the contact us section at the bottom of this page.

Project Goals

Before getting into the specifics of our implementation goals, there are three areas that we have strong goals for. These are User functionality, User data protection, and Reliability & Usability.
